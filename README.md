# polar-night-circadian-headgear
Open-source wearable circadian phototherapy gear for polar nights. Features an unnetworked microcontroller and an edge-lit brim diffuser directing a 5-stage variable-spectrum light sequence (460nm blue to 660nm red) down to lower retinal ipRGC receptors, providing distinct, non-continuous diurnal anchors using &lt;2Wh daily.
